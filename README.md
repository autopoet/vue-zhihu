### 项目概要

**CampusHub** 是专为高校极客打造的竞赛协作平台。通过极致的交互设计与底层性能优化，连接每一个有志于挑战竞赛、打磨产品的灵魂。

### 核心特性

- **Bento 仪表盘**: 现代便签盒布局，极致的可视化个人看板。
- **Web Worker 计算**: 独立线程处理高负载计算，UI 响应维持 60FPS。
- **玻璃拟态 UI**: 深层次视觉美学，沉浸式极客交互体验。
- **极客进化论**: 记录成长轨迹，打造独一无二的自动化档案。

### 技术矩阵

- **核心**: Vue 3.5+ (Composition API)
- **状态**: Pinia
- **构建**: Vite 6.x
- **布局**: Grid & Flexbox
- **离线**: Web Workers

### 快速启动

```bash
pnpm install
pnpm dev
pnpm build
```
